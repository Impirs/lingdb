# screenshots/

Snimci ekrana stvarnih terminalskih prolaza pajplajna — koriste se u `lingdb.ipynb`
(težak pajplajn se pokreće iz terminala, a notebook prikazuje rezultat + ove snimke).

Očekivana imena fajlova (referencirana iz notebook-a):

| fajl | prolaz |
|------|--------|
| `faza1_import.png`   | `python src/main.py --phase import`   |
| `faza1_morph.png`    | `python src/main.py --phase morph`    |
| `faza2_concepts.png` | `python src/main.py --phase concepts` |

Dodavanje: ili sačuvajte sliku ovde pod gornjim imenom, ili u notebook-u uđite u
markdown ćeliju (dvoklik) i **prevucite sliku** u nju — Jupyter će je umetnuti kao
prilog (`attachment:`).
